### Option 1: GitHub Pages (Simplest)
**Time: ~5 minutes**

```bash
# 1. Create GitHub repository at github.com
# 2. Clone and push your code
git remote add origin https://github.com/YOUR_USERNAME/cancer-awareness-support.git
git branch -M main
git push -u origin main

# 3. Go to Settings > Pages
# 4. Select "Deploy from a branch" > main > / (root)
# 5. Done! Your site is live at:
# https://YOUR_USERNAME.github.io/cancer-awareness-support/
```
